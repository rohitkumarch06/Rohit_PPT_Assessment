#### Task 20: Azure VNet (Basic Networking Practical)



In this task, I created a Virtual Network (VNet) in Azure and configured subnets for resource organization. A VNet allows secure communication between Azure resources. I also created and attached a Network Security Group (NSG) to control inbound and outbound traffic. NSG rules were configured to allow or deny specific ports. This task helped me understand basic networking concepts in Azure. It also improved my knowledge of cloud network security and traffic management.
