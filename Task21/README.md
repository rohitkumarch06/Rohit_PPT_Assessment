#### Task 21: Entra ID (Users, Roles, Basic Identity Management)



In this task, I worked with Azure Entra ID to manage user identities and roles. Entra ID provides centralized identity and access management for Azure resources. I created new users and assigned built-in roles based on required access. Role-based access control ensures users get only necessary permissions. This improves security and reduces risk of unauthorized actions. Through this task, I understood the basics of identity management and access control in Azure.
