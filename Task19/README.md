#### Task 19: Azure Storage Account (Blob, File, Table)



In this task, I created an Azure Storage Account and explored different storage services such as Blob, File, and Table storage. Blob storage is used for storing unstructured data like images and documents. File storage provides shared file system access. Table storage is used for storing structured NoSQL data. I uploaded sample data to understand how each service works. This task helped me understand different storage options available in Azure and their use cases.
